## Description
A robust command-line personal finance tracker CLI for adding, listing, summarizing, and exporting transactions with atomic JSON persistence at ~/.ledger/data.json. Supports income/expenses, filtering by month or category, colored tables, and manual CSV export without external libraries beyond Commander.js and uuid.

## Architecture
Modular CLI architecture separates command parsing in cli.js, atomic JSON storage in storage.js, and rendering/filtering utilities in utils.js. Commander.js parses arguments and invokes handlers that interact with storage via async functions. Production-ready with Jest tests mocking fs/process for 100% coverage, ES modules, strict mode, and cross-platform home dir handling.

## Tech Stack
- Node.js 20+
- Commander.js

- Jest

## Setup

npm init -y

npm pkg set bin.ledger=bin/ledger.js
